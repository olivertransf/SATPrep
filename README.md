# SATPrep

An iOS/iPadOS app for studying SAT questions with comprehensive filtering, progress tracking, and iCloud sync.

## Features

- **Smart Filtering**: Filter questions by program, module, primary class, skill description, difficulty, seen status, and Bluebook source
- **Progress Tracking**: Track which questions you've seen and answered correctly/incorrectly
- **Statistics**: View overall accuracy and breakdowns by category (module, difficulty, primary class, skill)
- **iCloud Sync**: Sync your progress across all your devices
- **Full Content Display**: View complete passages, questions, answer choices, and explanations without scrolling
- **MathML Support**: Proper rendering of mathematical notation and geometry symbols
- **Dark Mode**: Full dark mode support
- **iPad Optimized**: Responsive layout that works great on both iPhone and iPad

## Requirements

- iOS 17.0+ / iPadOS 17.0+
- Xcode 15.0+
- Swift 5.0+

## Setup

1. Clone the repository
2. Open `StudySAT.xcodeproj` in Xcode
3. Enable iCloud capability in Xcode (Signing & Capabilities tab)
4. Build and run on your device or simulator

## Usage

1. **Start a Quiz**: Select filters to customize your quiz, then tap "Start Quiz"
2. **Answer Questions**: Select an answer choice and tap "Submit" to see the explanation
3. **Navigate**: Use "Back" and "Next" buttons to move between questions
4. **View Stats**: Check the Stats tab to see your accuracy and progress
5. **Sync Progress**: Enable iCloud sync in Settings to keep your progress across devices
